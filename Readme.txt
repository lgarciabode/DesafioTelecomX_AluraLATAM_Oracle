📊 Análisis de Evasión de Clientes (Churn) - Telecom X

🔍 Descripción

Este proyecto tiene como objetivo analizar y comprender los factores que influyen en la evasión de clientes (Churn) en Telecom X. A través de un análisis exploratorio de datos (EDA), se identificaron patrones y variables clave que afectan la retención de clientes.

🧰 Tecnologías Utilizadas

Python: Lenguaje de programación principal.

Pandas: Manipulación y análisis de datos.

Matplotlib & Seaborn: Visualización de datos.

Jupyter Notebook: Entorno de desarrollo interactivo.

📈 Análisis Realizado
1. Distribución de la Evasión

Se analizó la distribución de la variable objetivo "Baja" para comprender la proporción de clientes que abandonaron el servicio.

2. Variables Categóricas

Se exploraron variables como "Género", "Método de Pago" y "Contrato" para identificar su relación con la evasión de clientes.

3. Variables Numéricas

Se examinaron variables como "Total Gastado", "Tiempo de Contrato" y "Cuentas Diarias" para detectar patrones asociados con la evasión.

4. Correlaciones

Se calculó la matriz de correlación para identificar relaciones entre variables y su impacto en la evasión.

🧠 Conclusiones

Contratos Mensuales: Los clientes con contratos mensuales presentan una mayor tasa de evasión.

Método de Pago: No se observó una relación significativa entre el método de pago y la evasión.

Gastos Elevados: Clientes con gastos totales más altos tienen una mayor probabilidad de abandonar el servicio.

🛠 Recomendaciones

Incentivar Contratos Anuales: Ofrecer beneficios para contratos de mayor duración.

Monitorear Clientes de Alto Gasto: Implementar estrategias de retención para clientes con altos gastos.

Mejorar el Soporte al Cliente: Fortalecer el servicio al cliente para reducir la tasa de evasión.
